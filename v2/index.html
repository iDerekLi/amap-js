AMapJS

基于AMap高德地图构建的API加载器。

快速上手 →

易用

开箱即用，最少的配置帮助你专注于开发。

灵活

更加友好的指定 API 版本。

模块化

支持按需引入，允许多次执行加载操作，网络资源不会重复请求。

# 快速开始

安装

# 通过 npm 安装
npm i amap-js -S

# 通过 yarn 安装
yarn add amap-js

使用AMap

// 创建AMap Loader
const loader = new AMapJS.AMapLoader({
  key: '您申请的高德Key值',
  version: '2.0',
  plugins: [],
});

// 调用 load 方法完成加载并调用执行回调。
loader.load()
  .then(() => {
    // 请求成功
    console.log(AMap); // => window.AMap === loader.AMap
  })
  .catch(e => {
    // 请求失败
  });
